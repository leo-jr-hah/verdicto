## Wallet Integration (CRITICAL)
- **Official Casper Wallet SDK** (injected by extension) — NO external npm packages needed
- `window.CasperWalletProvider` is the factory function (NOT `window.casper` which was deprecated)
- `window.CasperWalletEventTypes` has event names: Connected, Disconnected, ActiveKeyChanged
- Flow: `const provider = CasperWalletProvider()` → `provider.requestConnection()` → listen for events → `provider.getActivePublicKey()`
- `provider.isConnected()`, `provider.sign()`, `provider.signMessage()`, `provider.disconnect()`
- **CRITICAL: provider.on() does NOT exist!** Events are DOM CustomEvents on `window`:
  - `window.addEventListener(CasperWalletEventTypes.Connected, handler)`
  - `event.detail` is JSON: `{ activeKey, isConnected, isLocked }`
- **Payment flow:** `signPayment()` in CSPRClickContext creates a native transfer deploy via `casper-js-sdk`, asks wallet to sign it, returns base64 payment proof
- **Assessment fee:** 2.5 CSPR per assessment (shown in PaymentModal before running)
- Chrome Web Store: `https://chromewebstore.google.com/detail/casper-wallet/abkahkcbhngaebpcgfmhkoioedceoigp?hl=en`
- Testnet Faucet: `https://testnet.cspr.live/tools/faucet`
- WalletConnectButton is rendered in Layout.tsx sidebar footer + mobile header
- Faucet link is in sidebar above wallet button (not in dropdown)
- Reference implementation: https://github.com/make-software/casper-wallet-playground

## Payment Flow (Real On-Chain)
- `signPayment()` in CSPRClickContext: create deploy → wallet signs → **attempts** broadcast via `account_put_deploy` RPC → gracefully handles CORS failure
- RPC endpoint: `https://node.testnet.cspr.cloud/rpc` — **CORS blocks browser requests**, so broadcast often fails; signed deploy is still returned as cryptographic proof
- `broadcast: true/false` flag in payment proof indicates whether it was relayed on-chain
- Platform wallet: `02039cd256da1f2e13fc24a6f2ad1c15166f45070befa52bc2da46bbe194e7381010` (matches orchestrator .env DEPLOYER_PUBLIC_KEY)
- Shared config: `dashboard/src/config/casper.ts` — single source of truth for wallet address, fees, RPC URL
- Assessment fee: 2.5 CSPR, Prediction fee: 1 CSPR
- Explorer verification: `https://testnet.cspr.live/deploy/{hash}`
- **CORS note**: In production, backend would relay the signed deploy. For hackathon, the signed proof is sufficient.
