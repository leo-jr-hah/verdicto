## Wallet Integration (CRITICAL)
- **CSPR.click SDK** is the ONLY way to connect Casper Wallet v2.x
- `window.casper` does NOT exist — that was deprecated Casper Signer
- Packages: `@make-software/csprclick-core-client`, `@make-software/csprclick-core-types`, `@make-software/csprclick-ui`
- `signIn()` returns `void` — account arrives via `SIGNED_IN` event
- `signOut()` returns `void` — use synchronously
- `appId: 'csprclick-template'` works on localhost only
- Peer dep conflict: CSPR.click requires React 18, project uses React 19 — use `--legacy-peer-deps`
- `styled-components@5.3.9` installed as CSPR.click UI dependency
